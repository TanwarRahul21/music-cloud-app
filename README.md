# Music Cloud (HTML/CSS/JS)

A simple “music cloud” web app that stores your audio files in **your browser** (IndexedDB) and plays them like a modern music player.

## Project files

- `index.html`
- `style.css`
- `script.js`

## Run on localhost (step-by-step)

1) Open a terminal in this folder:

```bash
cd /home/rahul/JS
```

2) Start a local web server (Python is installed on your machine):

```bash
python3 -m http.server 8000
```

3) Open the app in your browser:

- http://localhost:8000

## How to use

1) Click **Add songs** (or drag & drop files into the drop area)
2) Uploaded songs appear automatically in the **Library** playlist
3) Click a song to play
4) Use **Play/Pause**, **Next**, **Previous**, and the **seek bar**
5) Use **Delete** to remove a song or **Clear library** to wipe everything

## Notes

- This app is client-only (no backend). Your songs stay in this browser.
- If you open the HTML file directly using `file://`, uploads/playback may behave inconsistently; use localhost.
