# Music Cloud App

A cloud-based music player where users can upload songs, store them online, and play them anytime from anywhere.

This project is built to understand real-world concepts like authentication, storage, API handling, and deployment.

---

## Features

- Upload and play music files  
- Cloud storage using Supabase  
- User authentication (Email / Google / Phone)  
- Personal music library  
- Audio player with controls (Play, Pause, Next, Previous)  
- Browser storage handling (IndexedDB)  
- Responsive UI  

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- Supabase (Auth + Storage + Database)  
- Vercel (Deployment)
